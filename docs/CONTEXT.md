# RoboRakshak Project Context

RoboRakshak is an AI-powered home security robot startup project designed for Indian households.

Core stack:
- Raspberry Pi 4 Model B
- L298N motor driver
- Flask backend
- Web-based control
- Pi Camera
- Battery-powered robotics platform

Current features:
- SSH remote development
- Web browser movement control
- GPIO motor control
- Battery-powered movement
- Flask-based WiFi robot control

GPIO Mapping:
- GPIO5 → IN1
- GPIO6 → IN2
- GPIO13 → IN3
- GPIO19 → IN4

Power setup:
- 12V Battery → L298N
- LM2596 → Raspberry Pi
- Shared GND between Pi and L298N

Upcoming goals:
- Camera streaming
- Motion detection
- AI surveillance
- Mobile app
- Face recognition
- Autonomous patrol
